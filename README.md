# Contador_blackjack

Este proyecto en principio consta de un juego desarrollado con windowsforms

La principal funcion es predecir las proximas cartas a salir en una baraja de BlackJack a traves de diferentes metodos de conteo

la idea de tener mas de un metodo de conteo es tener un menor margen de error y lograr obtener un mayor margen de acierto

Los metodos de conteo implementados son los siguientes:

- HiLo
- Omega
- Ko
- APM
- Wh
- Hi-2
- 4-4-5 (metodo de conteo ideado por mi)
